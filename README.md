# Blood Donation App 🩸

A full-stack MERN application that connects blood donors with people in need through location-based search and emergency request features.

---

## 🚀 Features

- 🔍 Search donors by blood group and location
- 📝 User registration and login
- 🚨 Emergency blood request system
- 📊 User dashboard
- 📱 Fully responsive UI

---

## 🛠 Tech Stack

### Frontend

- React
- Tailwind CSS
- React Router

### Backend (coming soon)

- Node.js
- Express.js
- MongoDB
